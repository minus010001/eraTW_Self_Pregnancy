ストーリー追加の際は表示条件、内容を埋めるだけではなく、ユーザ関数内の
K31_ストーリー進行　の通常処理部分の進行条件を弄る必要がある
例えば第1章第3節を追加したなら41行目
	IF CFLAG:31:1002 == 2　を
	IF CFLAG:31:1002 == 3　に変える必要があります
また、表示条件である指令の口上書き込み部分に
	CALL K31_特殊会話フラグの確認と出力("会話")
	SIF RESULT
		RETURN 1
といった文が必要（↑は会話によって表示する）
